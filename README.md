# 🛍️ Customer Behavior Analysis Project

## 📌 Overview

This project analyzes customer shopping behavior using **Python, SQL, and Power BI** to uncover insights that support data-driven business decisions. The workflow follows a structured pipeline from data preparation to final reporting.

---

## 📂 Project Structure

```id="projstruct01"
├── customer_behavior_of_shopping.ipynb   # Data preparation, cleaning & exploratory analysis (Python)
├── customer_behavior_analysis.sql        # SQL queries for business analysis
├── customer_behavior_dashboard.pbix      # Power BI dashboard for visualization
├── README.md                             # Project documentation
└── LICENSE                               # License file
```

---

## ⚙️ Workflow & Methodology

### ✅ 1. Data Preparation, Modeling & Exploratory Data Analysis (Python)

* Clean and preprocess raw data
* Handle missing values and inconsistencies
* Perform exploratory data analysis (EDA)
* Identify initial patterns and trends

📄 File: `customer_behavior_of_shopping.ipynb`

---

### ✅ 2. Data Analysis (SQL)

* Simulate business-related queries
* Analyze customer segments and behavior
* Evaluate customer loyalty and retention
* Identify key purchase drivers

📄 File: `customer_behavior_analysis.sql`

---

### ✅ 3. Visualization & Insights (Power BI)

* Build an interactive dashboard
* Visualize customer trends and KPIs
* Highlight key patterns in purchasing behavior
* Enable stakeholders to explore insights easily

📄 File: `customer_behavior_dashboard.pbix`

---

### ✅ 4. Report and Presentation

* Summarize key findings from the analysis
* Provide actionable business recommendations
* Create a presentation to communicate insights effectively
* Focus on clarity and decision-making impact

---

## 🚀 Getting Started

### 1. Clone the Repository

```id="clonecmd01"
git clone <repository-url>
cd <repository-folder>
```

### 2. Run Python Analysis

```id="runpy01"
jupyter notebook customer_behavior_of_shopping.ipynb
```

### 3. Execute SQL Queries

* Open `customer_behavior_analysis.sql` in your SQL environment
* Run queries to extract insights

### 4. Open Power BI Dashboard

* Launch Power BI Desktop
* Open `customer_behavior_dashboard.pbix`
